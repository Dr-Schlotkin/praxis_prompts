FAITHFULNESS_PROMPT = """
You are evaluating the FAITHFULNESS of a system answer using ONLY the provided CONTEXT.

Task:
- Identify the answer's key factual claims (events, actors, dates, causal assertions, numbers, quotes).
- For each claim, decide whether it is:
  SUPPORTED (explicitly stated or clearly implied by CONTEXT),
  CONTRADICTED (CONTEXT states the opposite),
  NOT_IN_CONTEXT (missing from CONTEXT).

Scoring (1–5):
5 = All key claims are SUPPORTED; at most 1 minor NOT_IN_CONTEXT detail.
4 = Core claims SUPPORTED; some minor NOT_IN_CONTEXT details; no contradictions.
3 = Mixed: at least one important claim NOT_IN_CONTEXT OR support is weak/ambiguous.
2 = Most key claims NOT_IN_CONTEXT OR at least one important contradiction.
1 = Largely hallucinated or contradicted; CONTEXT does not support the answer.

Rules:
- Use ONLY the CONTEXT; ignore your outside knowledge.
- If the answer adds specificity (dates, locations, numbers, named entities) not in CONTEXT, penalize.
- If CONTEXT is insufficient to verify, prefer NOT_IN_CONTEXT (do not guess).

Return ONLY JSON:
{"faithfulness": <1-5>, "short_explanation": "<one or two sentences naming the main supported/unsupported claims>"}
""".strip()
